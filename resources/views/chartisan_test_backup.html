Chartisan

Supercharged language agnostic charts

Getting started
Language Agnostic
Write it once and forget it
Written without a specific language or framework in mind. Build and configure your chart once, and expect it to work with all the supported backends.
Performant
Does not impact page load
The data is fetched in the background, allowing your application to perform as always does without slowing down the page loading time.
Customizable
Fully customizable to the core
Chartisan's hooks allow charts to be customized to the core, being able to control every aspect of the chart directly on the front-end library.
